xtomo_reader¶

Read image data from various format files.

This module provides low level file format support to xtomo_importer

Supported image fomats include TIFF, PackBits and LZW encoded TIFF, HDF5 (Data Exchange and NeXuS), HDF4 (NeXuS), SPE, TXRM, XRM, EDF, DPT, netCDF.

Author:Francesco De Carlo
Organization:Argonne National Laboratory, Argonne, IL 60439 USA
Version:2014.08.15

Requirements¶

  • h5py
  • netCDF (optional for supporting APS 13-BM data)
  • pyhdf (optional for supporting APS 2-BM data)
  • PIL.Image
  • Tifffile.c 2013.01.18 (optional for supporting Elettra data) (recommended for faster decoding of PackBits and LZW encoded strings)

Notes¶

PackBits and LZW encoded TIFF tested on little-endian platforms only.

Examples¶

>>> f = XTomoReader(_file_name)
>>> if (data_type is 'hdf4'):
>>>     tmpdata = f.hdf4(x_start=slices_start,
>>>                         x_end=slices_end,
>>>                         x_step=slices_step,
>>>                         array_name='data')
