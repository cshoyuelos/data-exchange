Welcome to Data Exchange’s documentation!¶

Data Exchange is a Python toolbox for reading and writing Data Exchange files. It also provides an interface between tomographic raw datasets collected at different Synchrotron Facilities and tomoPy, a Python toolbox to perform tomographic data processing and image reconstruction tasks developed at the Advanced Photon Source.
